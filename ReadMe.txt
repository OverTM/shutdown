更新说明

1.完善了基本的功能。

2.优化界面。

3.添加闹钟功能。


注意

1.重新启动时默认以关机进行响应。

2.关闭按钮只是最小化程序，并未退出。可以在托盘菜单或者程序菜单退出。

3.开机启动只是测试功能。每次重启该程序会清除；如果设置了开机启动，退出该程序依然可以开机启动。

4.如果使用者觉得该程序有什么问题，请及时摒弃，由该软件造成的后果与作者无关。作者未在该程序中种木马或后门等损害使用者利益的代码(可在源码查证)，但作者能力有限，程序可能存在纰漏。

5.该程序未使用算法或者设计模式，性能有点欠佳。不过涉及的知识还是很广的，很适合新手了解一些API或者MFC函数。

6.转载请注明作者，并保证文章的完整性。


已知BUG

1.资源管理器重启时，可能托盘图标无法自动重绘。

2.自定义闹钟音乐有时不能正常播放。遇到此情况，试着换音乐或者将程序以及音乐移动到其他文件夹。作者能力有限，调试很长时间无法修复，猜测的BUG代码在源码中已标出位置。

3.倒计时显示时间（使用者想定时的时间）与定时器实际时间（实际定时的时间）存在LONGLONG->UINT的转换，即有可能过大的显示时间还没倒计时到0，自动关机就动作了。所以定时不宜过大（根据数据类型推断三个月之内的定时不会出错）。想真正实现两个时间一致，可以定时器串接。
